# :book: app-deno

Api simples criada utilizando o Deno

## Como rodar
`deno run --allow-net src/app.ts`

## Rotas

| Rota                 | Descrição                |
| -------------------- | ------------------------ |
| GET /users           | retorna todos os usuários|
| POST /new            | adiciona um novo usuário |

> Que a força esteja com você :muscle:
